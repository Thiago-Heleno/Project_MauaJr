# PyQt5 Application for Maua Jr

## Installation

### Prerequisites

- Python 3.x
- PyQt5

You can install the required dependencies using `pip`:

```bash
pip install -r requirements.txt
```

or

```bash
pip install PyQt5
```

### Run the application

```bash
py app.py
```
